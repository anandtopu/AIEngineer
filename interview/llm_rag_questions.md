# LLM + RAG — Interview Questions

## LLM fundamentals

1) What is **tokenization** and why does it matter?

2) Explain **self-attention** in simple terms.

3) What is the **KV cache** and how does it affect latency?

4) What are common fine-tuning approaches (prompting, LoRA, full fine-tune)?

5) What is **hallucination**? How do you mitigate it?

## RAG

6) When would you use RAG vs fine-tuning?

7) What are the main components of a RAG system?

8) Chunking strategies: fixed size vs semantic; overlap tradeoffs.

9) Retrieval: dense vs sparse; hybrid search.

10) Reranking: why and how.

11) Evaluation: what metrics for RAG? (answer faithfulness, context relevance)

## LLMOps

12) How do you monitor quality in production?

13) Guardrails: prompt injection, data exfiltration, PII.

14) Cost controls: caching, batching, routing, smaller models.
