# Mock Interview Loop (Daily)

## 30–60 minute routine

1) Pick 5 questions from one topic file.

2) Answer each out loud:
- 2 minutes: high-level answer
- 3 minutes: deeper details + tradeoffs
- 1 minute: failure modes and how you’d validate

3) For 1 question, write a short solution as if in a real interview:
- assumptions
- approach
- complexity
- edge cases

4) Review:
- what you missed
- make a flashcard
- implement a small code exercise from `src/`
