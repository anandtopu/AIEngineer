# Module 01 — Python for ML (Interview-Oriented)

## Goals

- Be fluent with Python data structures and complexity
- Be comfortable with NumPy/pandas basics
- Be able to write clean, testable code under time pressure

## Topics to know

- Lists, dicts, sets, tuples
- Time complexity for common operations
- Iterators/generators
- `dataclasses`
- Type hints (basic)

## Practice

- Implement `top_k_frequent(nums, k)`
- Implement `group_by_key(pairs)`
- Read and run:
  - `src/basics/metrics_from_scratch.py`

## Interview prompts

- When is a `set` better than a `list`?
- Why can `dict` operations be O(1) average but O(n) worst case?
