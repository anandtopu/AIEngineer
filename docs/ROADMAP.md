# Roadmap (Beginner → Advanced)

This roadmap is optimized for AI Engineer interviews.

For a day-by-day schedule, use `docs/3_MONTH_PLAN.md`.

## Track A — Core Foundations (must-have)

- Module 01: Python for ML (data structures, NumPy, pandas)
- Module 02: Probability & Statistics for ML (distributions, expectation, variance)
- Module 03: Linear Algebra essentials (vectors, matrices, SVD intuition)
- Module 04: Supervised ML (linear/logistic regression, trees, ensembles)
- Module 05: Model evaluation (metrics, calibration, confidence intervals)
- Module 06: Data pitfalls (leakage, imbalance, shift, missingness)
- Module 07: SQL for AI Engineers (joins, grouping, window functions)

## Track B — Deep Learning (common for AI Engineer roles)

- Module 08: PyTorch fundamentals (datasets, dataloaders, training loop)

Recommended practice:
- `src/deep_learning/pytorch_training_loop_mnist.py`

## Track C — LLMs + RAG (increasingly expected)

- Module 09: LLM + RAG primer (RAG components, evaluation, tradeoffs)

Recommended practice:
- `src/rag/tfidf_rag_demo.py`
- `projects/02_rag_baseline/run.py`

## Track D — System Design + MLOps (differentiator)

- Module 10: ML system design primer
- Module 11: MLOps + LLMOps primer

Recommended practice:
- `projects/03_monitoring_drift/drift_check.py`

## What to do daily

- 60–120 min: learn + notes (`docs/`)
- 60–120 min: implement/run code (`src/`)
- 30 min: interview Qs (`interview/`)

## Suggested pacing

- 2 weeks: Track A + key parts of B
- 1 month: A + B + intro C
- 2–3 months: A + B + C + D (strongest)
